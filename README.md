## 🏋️ GymOfflineSystem

A lightweight and portable Gym Management System built with ASP.NET Core MVC (.NET 8) that works completely offline without requiring any database.

## ✨ Features
📁 Offline Data Storage (JSON-based)
👥 Client Management System
💳 Payment Tracking System
📊 Reports Generation
🔄 Automatic Year Handling
⚡ Auto Deactivation of Inactive Members
🖥️ Runs as Standalone EXE (No Database Required)

## 🛠️ Technologies Used
ASP.NET Core MVC (.NET 8)
C#
JSON File Storage
HTML, CSS, JavaScript

## ⚙️ How It Works
All data is stored in local JSON files
No need for SQL Server or any database
System automatically:
Creates yearly payment files
Updates system configuration
Manages inactive members

## 📂 Project Structure
Models → Data Models
Services → Business Logic (JSON handling, payments, reports)
Controllers → Application logic
wwwroot → Static files (CSS, JS)

## 🎯 Purpose

This project is designed for:
Small gyms without internet access
Offline-first systems
Easy deployment without database setup
